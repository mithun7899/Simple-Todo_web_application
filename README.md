# Simple-Todo_web_application
A clean, notebook-styled to-do list application built with Flask, HTML, CSS, and JavaScript. Designed to feel like writing on a paper notepad — simple, distraction-free, and functional.

📌 About The Project

The List is a lightweight to-do list web application with a warm, paper-notepad aesthetic — complete with a torn-tape header, ruled margins, and a typewriter-inspired font pairing (Courier Prime, Lora, JetBrains Mono).

The app lets users:


✅ Add new tasks
✅ Mark tasks as done / undone
✅ Filter tasks by All, Open, or Done
✅ Delete individual tasks
✅ Clear all completed tasks in one click
✅ See a live count of open vs. total tasks
✅ Persist tasks automatically between sessions


Built as a hands-on project to strengthen full-stack fundamentals — combining a Python/Flask backend with a hand-crafted, no-framework frontend.
